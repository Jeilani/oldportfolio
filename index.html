I first was able to become an expert in JavaScript through learning it in a processing environment, which allowed me to materialize animation design ideas. Keep clicking anywhere around the div for more balls to appear. (Works better on desktop). I'm also experienced in jQuery and have utilized a variety of its functionality.

Mini Projects

Air Jordan Project

Through the processing enviornment I was able to use object-oriented programming principles to allow a user to test the powers of air jordan himself to score as many points as possible while avoiding the blocking power of shaq in level 2.

I also heavily utizilized the power of conditional statements in scene changes

Trig Spaceship Project

Although not the most visually striking animation, this minitiaure spaceship flying through space required strategic brainstorming. and an ability to combine math and code/

It took a few minutes but I was able to tap deep enough into the principles of trigonometry and translate it into code. That combined with object oriented programming and conditional statements creates the rotating spaceship you see here. Check the source code and documentation to see.

Random Generator

Sometimes, randomness is good. Through processing, I was able to deliver a mountain range whose peaks and valleys are never repeated through code.

If you want to deliver variety within a piece of content, it's possible. It's as simple as soft-coding your content with variables that have random values everytime you load.
